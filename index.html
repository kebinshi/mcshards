Introduction

What in the world is MCShards?

MCShards is a free Minecraft hosting service focused on bring out the best player experiences and relive the Minecraft passion. Any player is allowed to create a vanilla server instance at no costs. Server settings, plugins, worlds can be changed and edited directly in game, eliminating the need for any external control panels. Plug and play!
